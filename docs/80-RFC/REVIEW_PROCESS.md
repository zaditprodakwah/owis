# RFC Review Process

## Review Stages

Draft
↓
Technical Review
↓
Architecture Review
↓
Community Review
↓
Approved / Rejected

## Evaluation Criteria

Architecture:
Does it preserve OWIS principles?

Compatibility:
Does it preserve v0.1.0?

Security:
Does it introduce new risks?

Adoption:
Does it improve external usability?

Complexity:
Is implementation justified?
