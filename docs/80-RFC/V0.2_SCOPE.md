# OWIS v0.2.0 Scope Proposal

## Included

The following proposals have passed architecture evaluation and are approved for implementation in v0.2.0:
- **RFC-001 WIR Graph Intelligence**: Additive graph modeling of workspace dependencies.
- **RFC-002 LLM Native Integration**: Context wrappers for AI frameworks to ingest WIR natively.
- **RFC-003 OWIS Schema Linter**: CLI command to audit workspace architecture quality.

## Deferred

Ideas requiring more validation before implementation:
- None at this time.

## Rejected

Ideas conflicting with OWIS principles:
- Proprietary agent framework built into OWIS (Conflicts with context-only boundary).
- Breaking changes to v0.1.0 WIR schema (Conflicts with backward compatibility rules).
