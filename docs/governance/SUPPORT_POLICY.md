# Support Policy

OWIS guarantees deterministic behavior, and its support lifecycle reflects our commitment to ecosystem reliability.

## Current Supported Versions
- **v0.2.x**: Active development and support. Receives all bug fixes, security patches, and minor feature updates.
- **v0.1.x**: Legacy. Receives critical security patches only until v1.0.0 is released.

## Long-Term Support (LTS)
Starting with OWIS v1.0.0, the project will establish an LTS program. LTS releases will be supported for a minimum of 18 months, during which they will receive security and critical bug fixes, but no new features.

## Alpha and Beta Releases
Alpha and Beta releases are strictly for preview and integration testing. They are **not** supported for production use and may contain unfixed bugs or unstable APIs. Issues reported against these versions will be addressed in the next release candidate, but patches will not be backported to previous alpha/beta tags.
