# OWIS Architecture Index

This document provides a machine-readable directory map of the Open Workspace Intelligence Specification (OWIS) architecture layers and their inter-document links.

---

## Constitution Layer

* [Project Constitution](../00-CONSTITUTION/Project_Constitution.md)
  * Defines the vision, core thesis, and design principles of the OWIS standard.
* [Documentation Governance](../00-CONSTITUTION/DOCUMENTATION_GOVERNANCE.md)
  * Defines documentation lifecycles, classifications, and change-control rules.
* [Release Governance](../00-CONSTITUTION/RELEASE_GOVERNANCE.md)
  * Defines version alignment rules and SemVer standards.

---

## Foundation Layer

* [Document Architecture Specification (DAS)](../01-FOUNDATION/DAS.md)
  * Defines the structure, layers, reading order, and governance of all documentation files.
* [Core Architecture Specification (CAS)](../01-FOUNDATION/CAS.md)
  * Defines the official conceptual model and domain ontology of workspace intelligence.
* [Reference Architecture Specification (RAS)](../01-FOUNDATION/RAS.md)
  * Defines reference components, data flows, and deployment topologies for the runtime engine.

---

## Specification Layer

* [Universal Agent Runtime Specification (UARS)](../10-SPEC/UARS.md)
  * Defines the runtime contract, pipeline stages, and operating guidelines for AI agents.
* [Workspace Intelligence Report Specification (WIR)](../10-SPEC/WIR.md)
  * Defines the data schemas representing the AI agent's synthesized workspace understanding.
