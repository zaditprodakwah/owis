# Roadmap

This document outlines the Go-to-Market and Phases for the OWIS standard.

## Phase 1
- Open Specification (UARS, WIR)

## Phase 2
- Schemas (JSON Schema, YAML Schema)

## Phase 3
- Reference Examples (Multi-language, Multi-framework, Multi-agent)

## Phase 4
- Reference Tooling (Validator, Generator, Inspector)

## Phase 5
- SDK & Integrations (TypeScript, Python, Go)
