# Contributing to OWIS

We welcome contributions to the Open Workspace Intelligence Specification!

## Getting Started

1. Read the Architecture Specifications in `/docs/10-SPEC/`.
2. Ensure any changes align with the Core Thesis and Vision.
3. Check the [Projects](https://github.com/zaditprodakwah/owis/projects) board to see what's currently being worked on.
4. Read our [Wiki](https://github.com/zaditprodakwah/owis/wiki) for additional contributor guides and patterns.
5. If you have an idea or question, start a thread in [Discussions](https://github.com/zaditprodakwah/owis/discussions).
6. To report a bug or request a feature, open an [Issue](https://github.com/zaditprodakwah/owis/issues).
7. Submit a Pull Request with a clear description of the proposed changes.

## Code of Conduct

Please adhere to standard open source guidelines and maintain a respectful environment.
