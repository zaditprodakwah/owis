# Medium Node.js API

A medium-scale Node.js REST API workspace used as a golden fixture for OWIS testing.

## Purpose

This fixture represents a typical Node.js project with multiple source files,
a package.json, and documentation. It is used to verify deterministic WIR generation
at medium scale.

## Structure

- `src/` — Application source files
- `src/index.js` — Entry point
- `src/routes.js` — API routes
- `src/models.js` — Data models
- `docs/` — Documentation
- `package.json` — NPM package manifest
- `workspace.json` — OWIS workspace configuration
