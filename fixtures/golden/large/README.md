# Large Full-Stack Application

A large-scale full-stack application workspace used as a golden fixture for OWIS testing.

## Purpose

This fixture represents a full-stack JavaScript project with frontend, backend, and shared
library code. It is used to verify deterministic WIR generation at large scale.

## Structure

- `frontend/` — React frontend application
- `backend/` — Node.js API backend
- `shared/` — Shared utilities and types
- `docs/` — Project documentation
- `package.json` — NPM workspace manifest
- `workspace.json` — OWIS workspace configuration
